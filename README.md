# Ionic Vue Starter-Kit (Skeleton)
Ionic Vue Starter Kit with features such as:
- [X] Admob
- [ ] FCM


### Tech Stack
- Ionic 5
- Vue 3
- Vue Router 4
- Capacitor 2


## Getting Started
**Serving the app**
```bash
ionic serve

#or 

ionic serve --no-open
```

**Building for production**

```bash
ionic build
```

**Copy assets to android**
```bash 
npx cap sync
```
